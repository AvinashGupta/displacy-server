About displaCy

displaCy lets you peek inside spaCy's syntactic parser, as it reads a sentence word-by-word. By repeatedly choosing from a small set of actions, it links the words together according to their syntactic structure. displaCy makes clever use of CSS and uses modern web standards to work with the browser, instead of against it.

Welcome to the new displaCy!
